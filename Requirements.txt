Requirements for this project:

1. Operating System: Windows
2. Version Control System: Git-based system (e.g., GitHub, GitLab)
3. CI/CD Tool: GitHub Actions / GitLab CI
4. Containerization Platform: Docker
5. Scripting Language: Python
6. Notification Service: Email
7. Web Browser: For monitoring pipeline execution

Non-Functional Requirements:

1. Reliability: The system should minimize manual intervention during failures.
2. Scalability: The system should support multiple projects and pipelines.
3. Performance: Pipeline execution overhead should be minimal.
4. Maintainability: Pipeline configurations should be modular and easy to update.
5. Security: Credentials and secrets must be stored securely.
